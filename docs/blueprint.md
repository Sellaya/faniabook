# **App Name**: FaniaBook

## Core Features:

- User Profile Management: Users can register/login and manage their profiles with contact details, preferences, and booking history using email, phone, or social login.
- Service Selection & Quote Generation: Users select from Bridal, Semi-Bridal, Party Makeup, and Classes, specifying in-studio or mobile services; a dynamic quote calculator estimates the price based on selected options, date, and location.
- Booking & Appointment System: Real-time calendar to book appointments, prevent double bookings, and sync confirmed appointments; users choose available time slots.
- Automated Notifications: Automated communication for quote confirmation, reminders, updates, and payment confirmations via email and WhatsApp; after order confirmation user and admin will receive emails of confirmation with booking details.
- Payment Integration: Secure online payment processing via Stripe; option for in-studio clients to pay in person on appointment day; mobile service requires 50% advance.
- AI-Powered Review Summary: Utilize an AI tool to summarize user reviews and feedback to help improve service quality. This would process and summarize customer reviews. This will function as a tool.
- Contract Generation: Create a contract before the user submits the payment

## Style Guidelines:

- Primary color: Teal (#008080) to evoke feelings of sophistication, trust, and tranquility, aligning with the professionalism of the makeup artistry.
- Background color: Light Beige (#F5F5DC) for a warm and inviting backdrop, providing a neutral canvas that complements the primary color.
- Accent color: Coral (#FF7F50) to highlight interactive elements, calls to action, and important information, adding a touch of vibrancy and energy.
- Headline font: 'Playfair', serif, for an elegant and fashionable look
- Body font: 'PT Sans', sans-serif, for clear and readable text; pairs well with Playfair
- Use elegant and clean line icons related to beauty and makeup services.
- Subtle, smooth transitions and animations for a modern and polished user experience.
